자신의 공간을 공유하고 **초대받으며** 같이 식사를 할 수 있도록 도와주는 플랫폼

## 기능

- 소셜 로그인
- 모임 개설, 검색, 참석
- 모임 리뷰 등록
- 프로필 사진 등록, 변경
- 실시간 채팅
